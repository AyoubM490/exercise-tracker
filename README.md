## Getting Started

* To get a local copy up and running follow these simple example steps.

1. clone the repository to you machine using your terminal:
   ```
   git clone https://github.com/AyoubMs/exercise-tracker
   ```
   ```
   cd exercise-tracker
   ```
2. Install the npm:
   ```
   npm install
   ```
4. To run the server use:
   ```
   npm start
   ```
